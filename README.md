# Hello World Program

 Overview
This software is my first small project as I continue learning software development.  
The goal of this program is to demonstrate that I can:

- Set up a working development environment
- Write a basic program that runs correctly
- Upload code to GitHub
- Record a short demonstration video

The program simply prints the text **"Hello World"** to the screen.

[Software Demo Video](https://www.loom.com/share/f140efbd8a254c27b895b1d3427e6e27)

 Development Environment
I used the following tools to develop this software:

- **Visual Studio Code (VS Code)** – to write and edit code
- **Python** – programming language used for this project
- **Git** – to upload the code to GitHub
- **GitHub** – to store and share the code online

 How to Run
1. Open the `hello.py` file in VS Code (or any text editor)  
2. Run the file using Python  
3. The program will display: Hello World
